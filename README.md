# Manutenção Industrial - Power BI
Este repositório apresenta um dashboard profissional desenvolvido em Power BI para análise de indicadores de manutenção no ambiente industrial. Os dados originais não foram incluídos por motivos de confidencialidade — aqui você encontra apenas a documentação e prints das telas.

## Objetivo do Projeto
No setor de Manutenção/Industrial percebiamos a deficiência de uma visão clara e detalhada sobre os indicadores do setor. Esse dashboard seviu para identificar as máquinas com mais tempo de indisponibilidade, cálculo da disponibilidade total de máquinas, índice de realização das preventivas, índice de comparação de horas paradas com o mês anterior em porcentagem, e tempo total de máquinas paradas. Com esse dashboard foi possível aprimorar o calendário de preventivas com relação aos anos anteriores.

## Tecnologias Utilizadas
- **Power BI Desktop;**
- Power Query para tratamento e modelagem dos dados;
- DAX para medidas e cálculos;
- Excel (como fonte de dados no projeto original).
